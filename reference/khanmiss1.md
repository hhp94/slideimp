# Khan microarray data with random missing values

A text file containing the Khan microarray data with random missing
values introduced for illustrative purposes. Adapted from the `impute`
package.

## Usage

``` r
data(khanmiss1)
```

## Format

The data set `khanmiss1` consists of genes on rows and samples on
column.

Please note that this dataset was derived from the original by
introducing some random missing values purely for the purpose of
illustration.

## Source

Khan, J. and Wei, J.S. and Ringner, M. and Saal, L. and Ladanyi, M. and
Westermann, F. and Berthold, F. and Schwab, M. and Antonescu, C. and
Peterson, C. and Meltzer, P. (2001) Classification and diagnostic
prediction of cancers using gene expression profiling and artificial
neural network. Nature Medicine 7, 673-679.

## References

Robert Tibshirani, Trevor Hastie, Balasubramanian Narasimhan, and
Gilbert Chu (2002). Diagnosis of multiple cancer types by shrunken
centroids of gene expression PNAS 99: 6567-6572. Available at
www.pnas.org
